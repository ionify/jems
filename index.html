jems: json-enhanced modules


jems are JSON and JavaScript Object Literals that implement the jem pattern for immediate-invocation, observation, and identification. The jem pattern is compatible with all ECMAScript Specifications and Engines. All jems are likewise compatible unless otherwise modified.

jems enable

  • Data and Functionality Encapsulation
  • Cross-Domain Data Exchange
  • Event-driven Programming
  • and Unobtrusive JavaScript

This repository hosts jem pattern documentation, its sample code, and the following trusted jems

  • jsonXD: json-extended dataflow
